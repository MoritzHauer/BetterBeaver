## Works today

Offline-first PWA; bundled content (Kyrgyz + demo topics) validated at startup; SM-2 review queue, streak, Topic → Lesson → Unit hierarchy with unlock gates at both levels (skippable via confirm) and lesson/topic-level practice shuffle; 10+ task types incl. sentence building, with cloze/build hint buttons; per-domain vocabulary screen with learner lists, shipped word families, ad-hoc study incl. dynamic 5-pair matching over saved words, TTS read-aloud; tap-to-lookup on post-answer session surfaces, vocabulary rows, and note views (best-effort resolve with add-word fallback), with hand-authored compound breakdowns in the entry popup; learner-created lexicon entries; per-domain review queue built from scheduling units (cloze blanks reviewed individually, grammar notes as self-graded flashcards, pinned tasks first) and streak; JSON export/import backup. All learner state persists in `localStorage`.

## Not yet built

- **User settings** — no settings screen, no persisted preferences.
- **Remote/selective content download** — all content is build-time bundled (fine at current ~500 KB); the `ContentSource` seam is where a remote source plugs in (`/ingest`, milestone 2 — will emit plan-0006 lexicon entries).
- **Play Store packaging** — PWA installs from the browser; TWA/Bubblewrap when a store listing is wanted.
- **Kyrgyz content backlog** — lexicon gaps surface via tap-to-lookup misses (plan 0008 points 4/5: e.g. a pronouns unit with мен/ал/сиздер as standalone entries), and cloze blanks that are names or inflected forms get no hint button until re-authored per the "don't blank names, prefer new concepts" guideline (plan 0008 point 11). Ongoing authoring work, not a discrete plan step.
