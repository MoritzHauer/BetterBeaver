## Works today

Offline-first PWA; bundled content (Kyrgyz + demo topics) validated at startup; SM-2 review queue, streak, Topic → Lesson → Unit hierarchy with unlock gates at both levels (skippable via confirm) and lesson/topic-level practice shuffle; 10+ task types incl. sentence building, with cloze/build hint buttons; a paginated Unit screen (dot-trail across Overview/Theory/Vocabulary/Concepts/Examples, per-note and chunked-overflow sub-pagers, swipe/keyboard navigation) with a sticky Practice bar that launches one pooled, shuffled session across the whole unit's tasks — task pinning now lives in that session itself, and Lesson/Topic/TopicList screens show graphical progress bars instead of plain text (plan 0010); per-domain vocabulary screen with learner lists, shipped word families, ad-hoc study incl. dynamic 5-pair matching over saved words, TTS read-aloud; tap-to-lookup on post-answer session surfaces, vocabulary rows, and note views (best-effort resolve with add-word fallback), with hand-authored compound breakdowns in the entry popup; learner-created lexicon entries; per-domain review queue built from scheduling units (cloze blanks reviewed individually, grammar notes as self-graded flashcards, pinned tasks first) and streak; JSON export/import backup. All learner state persists in `localStorage`. Visual identity: beaver mascot start screen, app-icon set, SVG UI icons (plan 0009). Auto-deploys to GitHub Pages on every push to `main` (`.github/workflows/deploy.yml`, served under `/BetterBeaver/`).

## Not yet built

- **User settings** — no settings screen, no persisted preferences.
- **Remote/selective content download** — all content is build-time bundled (fine at current ~500 KB); the `ContentSource` seam is where a remote source plugs in (milestone 3).
- **Play Store packaging** — PWA installs from the browser; TWA/Bubblewrap when a store listing is wanted.
- **Manual ingest backlog** — 13 of the Peace Corps manual's 15 lessons remain (plan 0007 scoped 7, shipped 2): 5 scoped with line ranges (Transportation, Bazaar, Post Office, Appearance, Weather) and 8 unscoped (Food, Apartment, Kiosk, Grocery, Department Store, Health, Daily Routine, At Work — each needs its own line-range pass first). Each is a run of the `/ingest` pipeline; per-lesson labor varies (see plan 0007's resolved questions).
- **Kyrgyz audio/picture tasks** — the task types ship (plan 0002) but no Kyrgyz unit uses them; blocked on the audio-production pipeline (TTS vs. recordings — open since plan 0002).
- **Kyrgyz content backlog** — lexicon gaps surface via tap-to-lookup misses (plan 0008 points 4/5: e.g. a pronouns unit with мен/ал/сиздер as standalone entries), and cloze blanks that are names or inflected forms get no hint button until re-authored per the "don't blank names, prefer new concepts" guideline (plan 0008 point 11). Ongoing authoring work, not a discrete plan step.
