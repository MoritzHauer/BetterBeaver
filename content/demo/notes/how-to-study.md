# How to read this lesson

Everything below is a real beaver fact wearing a disguise: it's also
explaining a piece of BetterBeaver. Study it exactly like any other unit —
the app doesn't know the difference between "the past tense of *jerlöö*" and
"why a beaver's teeth are orange," and neither should you.

A few things to notice as you go:

- **Recognize / Recall / Matching / Listen / Shadowing / Picture** all run on
  the five beaver-fact **concepts** in this unit — the same six task types
  any vocabulary-style content in the app uses.
- **Cloze / Dictation / Build** all run on one **sentence**, and
  **Scramble** runs on a second one — sentences are the other item kind, and
  cloze blanks in particular schedule and review individually, not as a
  whole sentence.
- **Minimal-pair** runs on a **pair** item ("dam" vs. "dad") — the one task
  type that exists purely for telling two similar sounds apart.
- This note itself is the eleventh content type: a plain explanation,
  reviewed later as a single self-graded flashcard, exactly like any grammar
  note.

Eleven task types, four item kinds, one note. That's the whole content
model — the rest of the app is just more of this, at scale, in whatever
subject you point it at.
