# Reading the rest of the alphabet

This unit drills the 19 letters you need first: the misleading ones plus all
the vowels (they drive vowel harmony). The rest of the 36-letter Kyrgyz
alphabet reads the way you would expect:

б = b, г = g, д = d, з = z, й = y (as in "yes"), л = l, м = m, п = p, т = t,
ч = ch, ш = sh

Four letters — ц, щ, ъ, ь — occur only in Russian loan words, and в (w) and
ф (f) are also mostly found in loans; you can leave all six for later.

Watch out for the look-alikes: Н is not H (it reads n), Р is not P (r), С is
not C (s), Х is not X (h), and У is not Latin y (it is the vowel u). The three
letters Ң, Ө, Ү exist in Kyrgyz but not in Russian.
